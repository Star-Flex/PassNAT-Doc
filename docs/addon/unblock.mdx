# 隧道解封

如果您的隧道因违规使用或违反用户使用准则而被封禁，因封禁行为是由信安设备自动检测或由相关部门通报封禁。
所以一般情况下不予以进行解封，甚至可能会导致账号被封禁。

不允许的一些行为可以参考以下链接，以下行为将不受理解封请求：

- [用户协议](https://www.passnat.com/illegal/)

特殊情况，比如

- 穿透的网站业务有漏洞需要修复
- 影响较小的特殊事件

可以通过提交工单的方式进行解封申请，同时客服人员可能会需要您 **手写** 保证书以留存相关证据。
保证书的相关内容及格式如下

```
本人隧道 (隧道ID) 因 xxx 被系统封禁。解封隧道后我将会严格遵守用户协议及相关规定，确保不再发生类似问题。
用户ID： xxxx
时间：xx年xx月xx日
```

隧道ID 可以在隧道列表中查看

用户ID 可以在账号管理-个人中心-用户识别ID 查看
