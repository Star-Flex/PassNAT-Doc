# 获取网站 SecretKey 密钥

> **注意**：SecretKey 密钥请妥善保管，不要泄露给他人，否则可能会导致账号被盗用等风险。

您的应用可能需要接入 PassNAT 提供的 API 服务，您需要先获取网站 SecretKey 密钥，具体步骤如下：

1. 登录访问 [PassNAT 控制台](https://dashboard.passnat.com) 进入个人中心页面。

   <img src={pic1} alt="1" />

2. 生成 SecretKey 密钥，点击生成 SecretKey 按钮即可生成。
