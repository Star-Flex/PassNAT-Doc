# 绑定 Bot 机器人

通过绑定 Bot 机器人，你可以直接在社区群里实现指令签到的功能，无需再通过网页端签到。

### 绑定 QQ 机器人

1. 首先前往控制台，点击左侧菜单栏的和账户管理，点击个人中心。
   或在右上角点击头像，选择个人中心。

   <img src={pic1} alt="1" />

2. 点击生成 access_token，access_token 只是用于绑定 QQ 机器人的密钥。泄露并不会造成账号损失。不过也需要及时进行绑定。

   <img src={pic2} alt="2" />

3. 复制 access_token，然后添加 QQ 群社区。

   <img src={pic3} alt="3" />

4. 一定要艾特机器人后再输入指令，否则机器人不会回复。

   <img src={pic4} alt="4" />
