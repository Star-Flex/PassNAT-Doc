# 邀请返利

PassNAT 提供邀请返利功能，用户可以通过邀请他人注册并购买套餐来获得返利奖励。

## 如何邀请

用户可以通过分享自己的邀请链接或邀请码来邀请他人注册。邀请链接可以在用户的 邀请返利-邀请好友 找到。
生成邀请码后需要把返利链接分享给好友进行注册，或者是让好友在注册时填写邀请码。

## 返利规则

- 好友注册后的每一单使用在线支付（支付宝/微信）时，邀请人将获得相应的返利奖励。不参与返利的支付方式包括：余额支付、礼品卡兑换。
- 返利比例可能会视运营情况而定，具体返利比例请参考 邀请返利-邀请好友 页面。
- 绑定好友的有效期暂定为 永久，后续可能会根据运营情况进行调整，具体以网站公告为准。

## 返利发放与提现

:::tip
特别注意：绑定的支付宝账户必须与 PassNAT 账号的实名认证信息一致，否则将无法通过提现校验。
:::

- 当绑定的好友成功支付订单后，返利金额将自动发放到邀请人的返利钱包中。用户可以在 邀请返利-返利 页面查看返利明细。
- 返利钱包的余额可以随时提现到绑定的支付宝账户。提现申请通常会在 1-3 个工作日内处理完成。并且按照国家相关法律法规定期代申报缴纳税款。
- 返利钱包的余额可以随时提现到网站余额钱包，提现申请通常会在即时到账。
