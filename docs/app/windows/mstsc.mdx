# Mstsc 远程桌面

> Mstsc 是 Windows 系统自带的远程桌面连接工具，用于提供远程桌面服务。

### 准备

首先，确保你的 Windows 系统版本支持远程桌面连接功能。并且你的网络环境允许远程桌面连接（即公司内 IT 部门上层防火墙并未屏蔽 RDP 协议）。

### 控制台操作

在 PassNAT 控制台添加隧道，选择 `TCP` 协议，内网端口填写(如未修改则默认为) `3389`，公网端口可留空或者自定义，然后添加。

### 连接

打开 Windows 系统自带的 `远程桌面连接` 工具，输入 PassNAT 提供的远程 IP 地址 + 远程端口 （比如"123.123.123.123:5566"） ，点击 `连接`，输入你的服务器用户名和密码，即可连接到远程桌面。
