# 常见问题

:::tip
本页面收集了一些常见问题，如果你在使用 PassNAT 时遇到了问题，可以在这里查找答案。建议使用 [PassNAT DocAI](/aipage) 输入关键词即可快速找到答案。本页面也主要充当语料供给 AI 使用。
:::

### 域名如何绑定

域名绑定分为两种情况，第一种情况是穿透 TCP/UDP 的基本协议。这种协议可以直接在 DNS 服务商使用 A 解析到 PassNAT 的 IP 地址即可，访问需要携带相应的端口号。
第二种情况是穿透 HTTP/HTTPS 的协议，这种情况需要在 PassNAT 控制台中绑定ICP备案域名，然后在 DNS 服务商使用 A 解析到 PassNAT 提供的公网 IP 地址。
同时如果你使用的是 PassNAT 提供默认的域名，可以直接使用，不需要再到 DNS 服务商进行解析。

:::tip 小技巧
在创建隧道时，你可以利用 HTTP/HTTPS 协议默认域名会自动添加解析的特点，也可以使用这个域名访问 TCP/UDP 服务，因为 HTTP/HTTPS 解析的 IP 地址和 TCP/UDP 解析的 IP 地址是一样的。只需要在域名后面加上 TCP/UDP 服务的端口号即可。
:::

### 访问隧道提示 "您访问的网站被拦截，可能是以下原因"

这是机房信安系统出现的拦截页面，出现这种报错一般有下面几种情况

1. 你直接访问了节点IP地址。如果你使用 TCP 穿透的隧道，你应该访问 节点:端口 的形式，而不是直接访问节点 IP 地址。如果你是使用 HTTP/HTTPS 穿透的隧道，你应该访问绑定的域名，而不是直接访问节点 IP 地址。
2. 你的备案域名没有添加到节点所在机房的白名单中，你应该在域名管理中点击 “过白名单” 按钮，选择对用的可用区进行添加。
3. 你的内容违反了国家法律法规，被机房信安系统拦截。你应该检查你的内容是否违规。

### 多个配置文件可以合并到一个配置文件吗

FRPC 原生就不支持多个配置文件合并到一个配置文件。但是你可以同时启动多个隧道。

### 添加的域名有什么要求

PassNAT 的域名添加有一些要求，首先你的域名必须是已备案的域名，这里的备案是指 ICP 备案，并不是域名的实名认证。其次你的域名必须是可以正常解析的域名，即你的域名在 DNS 服务商中可以正常解析到 IP 地址。

### 有些线路标记的禁海外，UDP是什么意思

因为网络上发生的攻击基本上来源于海外流量，且部分攻击基于 UDP 协议。

PassNAT 为了保证服务的稳定性，对一些线路进行了禁海外处理，这些线路在国内的访问更稳定，但是在国外的访问速度会受到影响。如果你的服务对象主要是国内用户，可以选择这些线路，如果你的服务对象主要是国外用户，建议选择其他线路。

### 远程/节点ip地址会更改吗

PassNAT 的节点全部使用 IDC 的线路资源。 IP 地址是固定的，不会更改。但某些极端情况下，可能会因为不可抗力因素导致节点 IP 地址更改，这种情况下我们会提前通知你。

### 线路有并发限制吗

PassNAT 的线路没有并发限制，但是有流量限制，如果你的流量用完了，你的隧道将会被暂停。

### 上下行流量计费是怎么计算的

PassNAT 的流量计费是按照上下行流量之和计算的，即上传流量和下载流量之和。但只是计算用户侧产生的流量，不计算内网终端与节点之间的流量。
举个例子。如果你的内网设备上搭建了一个网盘系统，你通过 PassNAT 的隧道上传了 1GB 的文件，然后又下载了 2GB 的文件，那么总共参与计费的流量是 2GB + 1GB = 3GB。内网终端与节点之间的流量不计算在内。

### 限速是限制上行还是下行

PassNAT 的限速会同时限速上行带宽和下行带宽，即上传和下载的速度都会受到限制。
举个例子。比如你目前的套餐端口限速是 100Mbps ，那么你的上行和下行带宽都会分别限制到 100Mbps。

### 什么流量会被计费

所有通过 PassNAT 隧道传输的流量都会被计费，包括上传和下载的流量。但只是计算用户侧产生的流量，不计算内网终端与节点之间的流量。

### 为什么我的多端口隧道只有一个端口能用

如果你在已有隧道中添加了端口，但只有一个端口或只有部分端口能用。这种情况下可能是你未更新配置文件信息，PassNAT 不会自动更新配置文件，你需要手动更新配置文件再重新启动隧道。

### 数据是加密的吗

PassNAT 不是所有的协议都是加密的。比如 HTTP 协议是明文传输的，HTTPS 协议是加密传输的。TCP/UDP 是最基础的传输协议，不会加密传输，但是你可以在应用层使用加密算法对数据进行加密。

### 套餐流量重置问题

PassNAT 的计费方式采用类似于运营商的计费方式，每个月的 1 号 0 点会重置流量。如果你的流量用完了，你的隧道将会被暂停，你可以选择购买额外的流量包，或者等到下个月流量重置后再使用。
因为从开发设计的角度来说，已使用流量与套餐标定的流量是相分离的。从原理上来看，套餐订阅的流量只是设置的一个阈 (yù) 值，当你使用的流量超过这个阈 (yù) 值时，你的隧道将会被暂停。而在每个月的 1 号 0 点，其实只是重新统计了此月已使用的流量，而非真实的给予新的流量。

举一个极端的例子：
比如你在 12 月 31 日 23:59:00 购买了包含 100G 流量的订阅，且在当天的 23:59:30 消耗了 99G 流量。那么在 1 月 1 日 0:00:00 时，因为触发了次月的流量重新统计机制，你会重新获得 100G 的可用流量。

### 一个月的流量如果用完了怎么办

如果你的流量用完了，你的隧道将会被暂停，你可以选择购买额外的流量包，或者等到下个月流量重置后再使用。

### 签到赠送的流量有效期多长

签到赠送的流量有效期是到月底，如果你在本月月底前没有使用完，剩余的流量将会被清空。次月签到赠送的流量会重新计算。

### 一个隧道可以绑几个域名

如果你使用的是 TCP 或 UDP 协议，那么一个隧道可以不限绑定域名，你只需做解析即可。如果你使用的是 HTTP 或 HTTPS 协议，那么一个隧道能绑定的域名数取决于你当前套餐所限制的隧道端口数。

### 网速很慢怎么办

网络速度慢和很多方面的因素有关，下面是几个常见的原因：

- 您本地的上行速度较慢，可以通过测速工具测试一下您的上行速度。如果测速正常，则有可能是运营商跨省限速 QOS 的问题。
- 您与 PassNAT 节点连接较慢，可以通过 Ping PassNAT 节点 IP 地址测试一下延迟。如果延迟较高，可以尝试更换线路。
- PassNAT 的节点负载较高，您可以在 PassNAT 控制台中查看节点负载情况，如果节点负载较高，可以尝试更换线路。

### 能免费试用吗

因全站使用BGP节点，节点成本较高。PassNAT 不支持免费试用，最低的套餐是 6 元/月，你可以选择购买这个套餐进行试用。

### PassNAT 实名认证是怎么样的

为满足国家相关法律法规要求，PassNAT 会对用户进行实名认证。实名认证需要提供真实姓名和身份证号码，您的实名流程会跳转到第三方可信平台（当前为支付宝蚂蚁金融人脸识别）进行实名认证。
我们只接收实名平台返回的认证结果。

### 如何重置密码

重置密码有两种方式，第一种是在已登录的情况下，你可以在控制台中的个人中心-账户密码进行修改密码。
第二种是在登录页面点击忘记密码，然后按照提示进行操作。

### 如何进行签到

目前有三个官方签到渠道，签到即可随机获得流量奖励。

1. 登录 PassNAT 控制台，首页会有签到按钮，点击签到按钮即可签到。
2. 登录 PassNAT 客户端，首页会有签到按钮，点击签到按钮即可签到。
3. 绑定 QQBot 机器人，加入官方 QQ 群按照机器人提示进行签到。

### 如何配置多端口映射

目前 PassNAT 已支持单隧道多端口功能，你可以在控制台中的隧道管理中添加端口，然后更新使用的配置文件即可。（如果使用 PassNAT 客户端则需要重新连接隧道才会生效）

# 财务类问题

### PassNAT 的订阅价格是多少

价格可能会存在变动，你可以登录控制台，在购买订阅中查看当前最新价格。

### 升级套餐

PassNAT目前还不支持跨套餐升级，请须知。如果需要更换套餐必须等原套餐到期后再进行更换。请不要担心，目前已创建的隧道在指定时间内不会因为订阅过期而被删除。具体时间可以通过工单或者加入 QQ 群询问。
不过你也可以退款当前的订阅，然后重新购买新的订阅。

### 怎么续费现有的订阅

你可以在 PassNAT 控制台中的财务中心-订阅管理页面进行续费，也可以通过工单或者加入 QQ 群询问。

### 退款问题 （更新时间：2024 年 12 月 28 日）

在你购买订阅的 24 小时内，你可以通过工单联系我们进行全额退款。超过 24 小时，我们将会按照使用时间进行退款。具体退款的金额可以通过工单确认。
但是有几个情况是不支持退款的：

- 你的账号被封禁，违反了 PassNAT 的使用规定
- 你使用的流量已超过订阅的 50%
- 流量包不支持退款
- 此订阅已申请增值税发票

#### 关于余额退款问题

PassNAT按订阅原渠道进行退款，如果你是通过支付宝支付的，我们会按照支付宝原订单原路退回。如果你是通过微信支付的，我们会按照微信原订单原路退回。如果你是通过余额支付的，我们会按照余额原路退回。
因余额充值与提现会存在被黑产利用收款的可能，PassNAT不支持余额提现，余额只能用于购买 PassNAT 的订阅。如有需要请尽量使用支付宝或者微信直接支付订单。

### 能开发票吗

PassNAT 提供增值税普通发票，可以在财务中心-发票申请页面申请开具。
