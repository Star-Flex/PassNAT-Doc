# 常见问题

### 域名如何绑定

域名绑定分为两种情况，第一种情况是穿透 TCP/UDP 的基本协议。这种协议可以直接在 DNS 服务商使用 A 解析到 PassNAT 的 IP 地址即可，访问需要携带相应的端口号。
第二种情况是穿透 HTTP/HTTPS 的协议，这种情况需要在 PassNAT 控制台中绑定ICP备案域名，然后在 DNS 服务商使用 A 解析到 PassNAT 提供的域名。
同时如果你使用的是 PassNAT 提供默认的域名，可以直接使用，不需要再到 DNS 服务商进行解析。

:::tip 小技巧
在创建隧道时，你可以利用 HTTP/HTTPS 协议默认域名会自动添加解析的特点，也可以使用这个域名访问 TCP/UDP 服务，因为 HTTP/HTTPS 解析的 IP 地址和 TCP/UDP 解析的 IP 地址是一样的。只需要在域名后面加上 TCP/UDP 服务的端口号即可。
:::

### 有些线路标记的禁海外，UDP是什么意思

因为网络上发生的攻击基本上来源于海外流量，且部分攻击基于 UDP 协议。

PassNAT 为了保证服务的稳定性，对一些线路进行了禁海外处理，这些线路在国内的访问更稳定，但是在国外的访问速度会受到影响。如果你的服务对象主要是国内用户，可以选择这些线路，如果你的服务对象主要是国外用户，建议选择其他线路。

### 远程/节点ip地址会更改吗

PassNAT 的节点全部使用 IDC 的线路资源。 IP 地址是固定的，不会更改。但某些极端情况下，可能会因为不可抗力因素导致节点 IP 地址更改，这种情况下我们会提前通知你。

### 线路有并发限制吗

PassNAT 的线路没有并发限制，但是有流量限制，如果你的流量用完了，你的隧道将会被暂停。

# 财务类问题

### 升级套餐

PassNAT目前还不支持跨套餐升级，请须知。如果需要更换套餐必须等原套餐到期后再进行更换。请不要担心，目前已创建的隧道在指定时间内不会因为订阅过期而被删除。具体时间可以通过工单或者加入 QQ 群询问。

### 一个月的流量如果用完了怎么办

如果你的流量用完了，你的隧道将会被暂停，你可以选择购买额外的流量包，或者等到下个月流量重置后再使用。

### 怎么续费现有的订阅

你可以在 PassNAT 控制台中的财务中心-订阅管理页面进行续费，也可以通过工单或者加入 QQ 群询问。

### 能开发票吗

PassNAT 提供增值税普通发票，可以在财务中心-发票申请页面申请开具。
